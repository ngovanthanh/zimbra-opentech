Définir Zimbra Desktop comme messagerie par défaut

Lorsque vous cliquez sur un lien de mail dans une application ou dans une page Web, le système lance votre messagerie par défaut.  

Vous pouvez demander que Zimbra Desktop soit utilisé comme messagerie par défaut. Dans ce cas, lorsque vous cliquez sur un lien de mail dans une application ou dans une page Web, le système ouvre une fenêtre de message vide dans la messagerie déclarée comme valeur par défaut dans Zimbra Desktop.

Procédez comme suit :  

  1. Sélectionnez Préférences générales.

  2. Dans la section Autre, notez la zone Valeurs par défaut du système. Dans cette zone, cochez la case Utiliser Zimbra Desktop comme application de messagerie par défaut.
