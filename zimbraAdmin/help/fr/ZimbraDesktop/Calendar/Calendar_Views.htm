Options d'affichage du calendrier

Vous pouvez configurer votre affichage de calendrier pour afficher vos jours et heures de travail.  Depuis la page Préférences > Calendrier, vous pouvez configurer :

  • La vue par défaut qui devrait être utilisée à l'ouverture de votre calendrier. Les vues sont les suivantes :

    • L'affichage Jour correspond aux activités d'une journée

    • La vue Semaine ouvrée affiche les activités du lundi au vendredi

    • La vue Semaine affiche les  activités de toute la semaine Dans l'onglet Calendrier, sous Préférences, vous pouvez sélectionner le premier jour de la semaine

    • La vue Mois affiche les activités d'un mois

    • La vue Programme affiche chaque calendrier dans une colonne séparée.

  • Le premier jour de votre semaine de travail.
