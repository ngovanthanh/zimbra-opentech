Accepter l'accès à un dossier de mails partagé

Remarque : Disponible pour les comptes de la société Zimbra uniquement.

Si vous recevez une notification par mail indiquant que vous êtes autorisé à partager le dossier de messagerie d'une autre personne, vous pouvez accepter ou refuser le partage.

Si vous acceptez le partage, le dossier de messagerie partagé s'affiche dans la liste de vos dossiers dans le panneau Aperçu. Lorsque vous travaillez sur Zimbra Desktop, pour voir les contacts des dossiers partagés, vous devez être connectés à Internet. Le contenu des dossiers partagés n'est pas téléchargé vers votre ordinateur.

Pour accepter

  1. Cliquez sur Accepter le partage dans le mail. La boîte de dialogue Accepter le partage s'affiche et décrit le rôle qui vous a été attribué.

  2. Avant d'accepter, vous pouvez changer le nom du dossier de mails et choisir une couleur pour contraster le dossier.

  3. Facultatif : choisissez d'envoyer une confirmation au cédant.

  4. Cliquez sur Oui. Le dossier de messagerie est ajouté à votre liste de dossiers. Ce message est transféré vers le dossier Corbeille.
