Déplacer des mails dans des dossiers

Vous pouvez transférer des mails ou des conservations complètes d'un dossier vers un autre :

  • Faites glisser l'élément et déposez-le dans le dossier souhaité dans le panneau de gauche.

  • Sélectionnez les éléments, cliquez avec le bouton droit et choisissez Déplacer. Dans le champ Déplacer le mail, choisissez le dossier de destination, puis cliquez sur OK.

Si vous déplacez des mails d'une conversation, la conversation apparaît dans les deux emplacements. La fenêtre Affichage des détails de la conversation indique le nouvel emplacement du mail qui a été déplacé.

Remarque : Si vous transférez un message vers un dossier, vous pouvez immédiatement annuler cette action en cliquant sur le lien Annuler qui apparaît en haut de la page. Le message revient dans son emplacement d'origine.
