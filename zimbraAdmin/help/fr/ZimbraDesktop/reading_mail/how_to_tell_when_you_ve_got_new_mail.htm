Notification au sujet des nouveaux messages

Vous recevez de nouveaux mails au moment où vous vous connectez, à intervalles réguliers configurés, et lorsque vous le décidez, en cliquant sur Actualiser la réception du courrier.

Vous pouvez définir la fréquence à laquelle votre compte vérifie les nouveaux mails à partir de Préférences>Mail.  Par défaut, la recherche des nouveaux mails s'effectue toutes les 5 minutes.

Vous pouvez activer les alertes Desktop pour une notification de nouveau mail. Vous pouvez définir vos préférences sur la page >Préférences Mail. Les options d'alerte sont les suivantes :

  • Émettre un son. Un bip vous avertit lorsque vous recevez un mail dans votre boîte de réception.  Si vous avez coupé le son de votre ordinateur, aucun bip n'est émis.

  • Activer l'onglet Mail. Si vous n'êtes pas en train d'utiliser l'onglet Mail, ce dernier est mis en évidence lorsque vous recevez un mail dans votre boîte de réception.

  • Afficher brièvement le nom du navigateur. Lorsque vous recevez un mail dans votre boîte de réception, le navigateur clignote jusqu'à ce que vous ouvriez l'onglet Mail.

Vous pouvez repérer les mails non lus en recherchant les dossiers contenant un nom en gras et un nombre entre parenthèses à côté du nom du dossier. Par exemple, Boîte de réception (22) signifie que les conversations dans votre boîte de réception contient 22 mails non lus.
