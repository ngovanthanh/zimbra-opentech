Contrôle de la version des fichiers du Porte-documents

Le contrôle de version vous permet de gérer les changements apportés aux fichiers du Porte-documents. Le numéro de la version est affiché dans la liste du document. En affichant l'historique de version, vous pouvez consulter les anciennes versions du fichier et l'auteur. Vous pouvez restaurer une ancienne version du fichier, ou supprimer celles dont vous n'avez plus besoin.

Remarque : Chaque version d'un fichier est comptabilisée dans votre quota de compte.  

Les fichiers que vous avez envoyés peuvent être extraits afin d'empêcher que plus d'une personne travaille sur le même fichier du Porte-documents. Ces fichiers sont alors téléchargés sur votre bureau. En envoyant le fichier, il est enregistré et la version est modifiée.
