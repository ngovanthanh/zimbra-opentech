Créer des listes de tâches

Vous pouvez créer différentes listes de tâches et configurer les listes pour synchroniser les listes de tâches à partir de services à distance, tels que la liste des tâches iCal.

Pour créer une liste de tâches :

  1. Cliquez sur l'onglet Tâches pour ouvrir l'application Tâches.

  2. Dans le panneau d'aperçu, cliquez avec le bouton droit sur l'en-tête Tâches, puis sélectionnez Nouvelle liste de tâches.

  3. Dans la boîte de dialogue Créer une liste de tâches, saisissez le nom de la liste de tâches puis choisissez la couleur à utiliser pour afficher la tâche.

  4. Cochez Synchroniser les tâches à partir du service distant pour synchroniser la nouvelle liste avec une liste de tâches iCal. Saisissez l'URL iCal.

  5. Cliquez sur OK. La liste est ajoutée aux tâches dans le panneau d'aperçu.
