Zoekopdrachten opslaan

Als je een zoekopdracht maakt die je mogelijk later opnieuw wilt gebruiken, kun je deze opslaan.

  1. Maak de zoekquery door tekst in te voeren op de balk Zoeken of door een query te maken via Geavanceerd zoeken.

  2. Als je de zoekopdracht wilt opslaan, klik je op op de balk Zoeken. Het dialoogvensterZoekopdracht opslaan wordt geopend.

  3. Geef je opgeslagen zoekopdracht een naam. De zoekopdrachten worden standaard opgeslagen als een zoekmap in het gedeelte Zoekopdrachten van het deelvenster Overzicht, maar je kunt ook een andere map selecteren.

  4. Klik op OK. Je zoekopdracht wordt opgeslagen.

Als je een opgeslagen zoekopdracht wilt gebruiken, klik je op de map Zoeken. De zoekresultaten worden onmiddellijk weergegeven in het deelvenster Inhoud.
