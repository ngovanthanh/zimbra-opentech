About the Auto Grouped Backup Method

If you use the auto-grouped backup method, update the backup mode either in the Global Settings Backup/Restore tab or in the Server's Backup/Restore tab to select auto-grouped backup mode and configure the number of groups that you have set up for the auto grouped backup.

Throttle automatic backups should be enabled when you are mass migrating new accounts to ZCS and use the auto-grouped backup method. The auto-grouped backup method automatically backs up mailboxes that have never been backed up when the next backup is scheduled.

Important: When migrating a large number of mailboxes, enable Throttle automatic backups so that the mailboxes that are migrated to ZCS are not automatically backed up as new mailboxes because the number of new mailboxes could prevent backing up existing mailboxes. Leave the throttle mode on for 7 days (cycle-through) after the mass migration is over. Allow the total mailbox count to reach an equilibrium, then let the auto-group backup cycle through before turning off the throttle mode.

The auto-grouped backup schedule can only be set from the CLI using the zmschedulebackup command.
