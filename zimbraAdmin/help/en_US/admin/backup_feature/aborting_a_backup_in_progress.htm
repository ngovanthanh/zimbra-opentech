Aborting a Backup in Progress

You can stop a backup session that is in progress by clicking Abort on the Backups toolbar.

Procedure

  1. Go to Tools and Migration>Backups. The servers are listed in the Content pane. The Status column shows whether a backup is progress or completed. If a backup is not complete, click on that server to open that server's Backup Session list. This shows the latest backup sessions and the status at the time.

  2. Select the backup that is in progress.  In the gear icon select Abort. The backup is immediately stopped.

  3. Click refresh and the session status is now marked as Competed with Errors.
