Starting a Standard Backup Session

When Zimbra Collaboration Server was installed, a backup schedule was automatically added to the cron table. The default full backup is scheduled for 1:00 A.M. every Sunday. The default incremental backups are scheduled for 1:00 A.M. Monday through Saturday.

You can also run the backup at any time from Backups. You can run the following backup sessions on the mailbox server

  • Full backup of all accounts

  • Incremental backup of all accounts

  • Full backup of selected accounts

Full backup is usually run as asynchronous.  The backup continues in the background. The Status column shows if a backup is running or has ended. Click Refresh to update the Status column.

  • If a backup session fails, an email message is sent to the configured administrator's email address.

To start a backup session

  1. Go to Tools and Migration>Backups.

  2. In the gear icon, click Backup. The Immediate Backup dialog box opens.

  3. Select the source server from the drop-down list.

  4. Select whether to run a Full or Incremental backup.

    • A full backup process backs up all the information needed to restore mailboxes, including the LDAP directory server, database, index directory, and message directory for each mailbox.

    • An incremental backup process backs up the LDAP directory server and gathers all the redo log transactions written since the last incremental backup or the last full backup, if this is the first incremental since the last full backup.

  5. The path to save the backup session is /opt/ zimbra/backup. The default directory for the backups is on the same mail server that is being backed up.  Unless you are saving to a separate disk drive, you can accept the default. Click Next.

  6. To backup all accounts on the server, check Backup all accounts. Click Backup.

To backup specific accounts, in the text box, enter an account name and click Search. The name (s) that match appear in the left column. Select the accounts to back up from that column and click Add or Add all to move the names to the right column. Click Backup.  

If you selected to run an Incremental backup, all accounts are backed up.

  1. The Running the backup dialog displays the session label. Click Finish to close the dialog.  The backup session continues in background.
