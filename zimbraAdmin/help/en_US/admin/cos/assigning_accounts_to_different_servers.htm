Assigning Accounts to Different Mail Servers

In an environment with multiple mail servers, the Class of Service is used to assign the next account to a mailbox server or you can specify a specific server when you create the account.

In the COS, the Server Pool lists the available mailbox servers. you can limit the servers available to the pool for the COS. When a COS is assigned to a new account, a mail server is randomly chosen for the new account from the server pool list associated with that COS.

Add Specific Servers to the Server Pool

  1. Go to Configure>Class of Service.

  2. Double-click on the COS to edit and select edit the Server Pool page.

  3. On the Server Pool page, add a check mark to the servers that should be available for creating new users.

  4. Click Save.

Note: If you select Do not limit server pool available for creating new users in this COS, all servers are in the server pool.

Add a Specific Server When Creating a New Account

  1. Go to Manage>Accounts.

  2. In the gear icon, select New.

  3. On the General Information dialog box, in the Server field uncheck auto.

  4. From the drop-down list, select the mail server for the new account.

  5. Continue to create the account and click Finish.
