Delegated Administration Target Types

A target is a Zimbra object on which rights can be granted on. The following are the specific types of targets that can be specified: Account, Calendar resource, Class of Service (COS), Distribution List (DL), Domain, Global Config, Global Grant, Server, Zimlet. See the Delegated Administration chapter in the ZCS Administration Guide for more information about target types.

 

Target Type

Description of the Target Scope

Account

An account entry (a specific user).

Calendar Resource

A calendar resource entry.

COS

Class of Service entry.

Distribution List

If the right is applicable to distribution lists, the distribution list and all distribution lists under this distribution list.

If the right is applicable to accounts and calendar resources, all accounts and calendar resources that are direct or indirect members of this distribution list.

Domain

Domain entry is only applicable to a specific domain, not to any sub-domains. Sub-domains must be explicitly marked as targets.

When domain is the target, the rights granted can be given for all accounts, calendar resources and distribution lists in the domain.

Config

Grants specific to global config.

Global

The global Access Control List (ACL) is used to grant admin rights for all entries in a target type. For example, creating accounts in all domains or manage all domains in the global environment.

Server

The server entry.

Zimlet

A Zimlet entry.
