Adding a New Storage Volume to the Server

Each Zimbra mailbox server is configured with one primary message volume and an index volume. Additional secondary message volumes can be added.

Before adding a new volume, the directory path must already exist.

Procedure

  1. Go to Configure>Server.

  2. Select the server and in the gear icon click Edit.

  3. Open  to the Volumes page and click Add.  

  4. Enter the volume name, root path for the new volume, and select the volume type:  

    • Primary Message

    • Secondary Message

    • Index

  1. If you want to compress blobs, check Compress Blobs and set the compression threshold.

    • Compress Blobs. When this box is checked, message blobs whose size is above the compression threshold are compressed. If blob compression is turned on, the disk space used is decreased. Note:  Turning on blob compression increases memory requirements for the server.

    • Compression Threshold. Messages larger than the threshold are compressed. The default threshold is 4096 bytes.

     

  2. Click OK.
