Сохранение запросов поиска

Если вы создаете профиль поиска, который будет использоваться в будущем, его можно сохранить.

  1. Создайте запрос поиска, введя текст на панели Поиск или создав запрос с помощью функции расширенного поиска.

  2. Чтобы сохранить профиль поиска, щелкните значок на панели Поиск. Откроется диалоговое окно «Сохранить профиль поиска».

  3. Введите имя для сохраненного профиля поиска. Профили поиска сохраняются по умолчанию в виде папки поиска в разделе Профили поиска в области просмотра, но можно выбрать другую папку.

  4. Нажмите кнопку ОК. Профиль поиска сохранен.

Чтобы воспользоваться сохраненным профилем поиска, щелкните папку профилей поиска. Результаты поиска сразу же отображаются в области содержимого.
