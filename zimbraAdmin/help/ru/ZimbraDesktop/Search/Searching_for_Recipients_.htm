Поиск сообщений, отправленных определенным получателям

Поиск сообщений по признаку того, кому они были отправлены, можно выполнять одним из следующих способов.

  • В папках «Отправленные» или «Черновики»

  • С помощью расширенного поиска, простого поиска в поле «Кому/Копия»

При выполнении поиска в какой-либо папке отображаются только сообщения, найденные в этой папке.  При использовании функции расширенного поиска можно выбирать папки, в которых необходимо выполнить поиск. В результаты будут включены сообщения из папок, выбранных для поиска.

Поиск в папках «Отправленные» и «Черновики»

  1. В папках Отправленные и Черновики выберите сообщение с именем получателя, которого необходимо найти.

  2. В заголовке щелкните поле Кому. Откроется диалоговое окно выбора адреса.

  3. Нажмите кнопку ОК. Выполняется поиск, и сообщения, отправленные данному получателю, отображаются в области содержимого.

Использование расширенного поиска

  1. Выберите пункт Дополнительно, чтобы открыть область расширенного поиска.

  2. В диалоговом окне Простой поиск в поле Кому/Копия введите имя для поиска.  Чем точнее имя, тем лучшими будут результаты поиска. По умолчанию поиск выполняется в папке «Входящие».

  3. Для поиска в других папках нажмите кнопку Папка и выберите папки, в которых необходимо выполнить поиск.

После ввода критериев в области поиска или после выбора элементов в другой области поиска будет выполнен поиск.

Элементы, соответствующие критериям поиска отображаются внизу справа. Если включена функция «Разговор», отображается список разговоров, которые содержат сообщения, соответствующие критериям поиска.
