Использование HTML-редактора

Сообщения можно создавать с использованием форматированного текста HTML.  Сообщения, созданные в формате HTML, могут содержать форматированный текст, маркированные и нумерованные списки, цветной фон, таблицы и ссылки, что может облегчить чтение сообщений.

Примечание. Некоторые почтовые клиенты могут не принимать сообщения с HTML-форматированием.

Можно установить HTML-редактор редактором по умолчанию, а также установить параметры шрифта по умолчанию, включая начертание, размер и цвет, на странице Параметры создания сообщений веб-клиента.
