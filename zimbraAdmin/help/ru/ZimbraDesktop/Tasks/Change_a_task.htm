Изменение задачи

Обновление задачи

  1. Откройте список задач с нужной задачей.

  2. В области содержимого щелкните задачу, которую нужно изменить, или выберите задачу и нажмите кнопку Изменить на панели инструментов. Откроется страница задачи.

  3. Внесите изменения в информацию о задаче.

  4. После завершения нажмите кнопку Сохранить.

Пометка задачи завершенной

Существует два способа обозначить задачи как завершенные:

  • Установите значки возле одной или нескольких задач, которые необходимо обозначить как завершенные. Щелкните правой кнопкой мыши строку с задачей и выберите пункт Пометить как завершенные. % завершения сменится на 100%.

  • Установите значки возле одной или нескольких задач, которые необходимо обозначить как завершенные. ЩелкнитеПометить как завершенные на панели инструментов.

Сведения о задаче не будут удалены из папки задач.
