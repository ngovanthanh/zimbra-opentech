Принятие доступа к общей папке почты

Примечание. Только для учетных записей Zimbra.

Получив почтовое уведомление о предоставлении общего доступа к папке почты другого пользователя, вы можете принять или отклонить профиль общего доступа.

Если вы соглашаетесь на общий доступ, общая папка почты будет отображаться в списке папок в области обзора. Чтобы видеть содержимое общих папок при работе в Zimbra Desktop, необходимо подключение к сети Интернет. Содержимое общих папок не загружается на компьютер пользователя.

Принятие профиля

  1. Щелкните ссылку Принять профиль общего доступа в почте. Откроется диалоговое окно «Принятие профиля общего доступа», описывающее предоставленную вам роль.

  2. Прежде чем принять профиль, можно изменить имя папки почты и выбрать цвет для выделения папки.

  3. Отправьте подтверждение распорядителю (необязательно).

  4. Нажмите кнопку Да. Папка почты будет добавлена в список ваших папок. Сообщение будет перемещено в Корзину.
