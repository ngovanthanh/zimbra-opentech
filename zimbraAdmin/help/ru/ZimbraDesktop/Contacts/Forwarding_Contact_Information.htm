Пересылка контактных данных

Контакт можно переслать из адресной книги в виде электронной визитной карточки, называемой vCard. vCard — это электронный эквивалент бумажной визитной карточки.

  • Выберите имя контакта и щелкните его правой кнопкой мыши. Выберите пункт Переслать контакт.

  • Выберите в адресной книге несколько контактов и на панели инструментов адресной книги нажмите кнопку Переслать.

  • В сообщении щелкните правой кнопкой мыши имя в поле «От» и выберите пункт Переслать контакт.

Откроется область создания сообщения с вложенным файлом .vcf контакта. Заполните и отправьте сообщение.  Получатель может загрузить файл .vcf непосредственно в свой список контактов.
